Google Chrome freeware web browser

Google Chrome is a freeware web browser developed by Google. It was first released in 2008 for Microsoft Windows, and was later ported to Linux, OS X, iOS and Android. Google Chrome is also the main component of Chrome OS, where it serves a platform for running web apps.

Mozilla Firefox beautiful fox

Mozilla Firefox is a free and open-source web browser developed by the Mozilla Foundation and its subsidiary, the Mozilla Corporation.

Safari best choice for Mac's user

Safari is a web browser developed by Apple based on the WebKit engine. First released in 2003 with Mac OS X Panther, a mobile version has been included in iOS devices since the introduction of the iPhone in 2007. It is the default browser on Apple devices. A Windows version, now discontinued,[4] was available from 2007 to 2012.

Opera small and easy

Opera is a web browser developed by Opera Software. The latest version is available for Microsoft Windows, OS X, and Linux operating systems, and uses the Blink layout engine. An earlier version using the Presto layout engine is still available, and additionally runs on FreeBSD systems.

IE old but popular

Internet Explorer[a] (formerly Microsoft Internet Explorer and Windows Internet Explorer, commonly abbreviated IE or MSIE) is a discontinued series of graphical web browsers developed by Microsoft and included as part of the Microsoft Windows line of operating systems, starting in 1995. It was first released as part of the add-on package Plus! for Windows 95 that year. Later versions were available as free downloads, or in service packs, and included in the original equipment manufacturer (OEM) service releases of Windows 95 and later versions of Windows.
